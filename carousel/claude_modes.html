3 Permission Modes — Which One to Use?

Mode What It Does Best For
🔍
Plan Mode
Read-only
 Claude can analyze and propose changes but cannot execute anything. Thinking mode only. Reviewing what Claude wants to do before it does it. Architecture planning.
✏️
Accept Edits I used this
Recommended
 Claude can read files, write files, and edit files without asking. Still asks permission for Bash/shell commands. Normal development flow — you trust file edits but want to approve commands.
⚡
Bypass Permissions
Use with caution
 Claude executes everything without asking — file edits, bash commands, network operations. Full autonomy. Experienced users in controlled environments only.
